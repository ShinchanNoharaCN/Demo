What's ICE Shader

ShinchanNohara

Dev Shader

【Illusion Code Effect Shader】

It is a shader program obtained after reverse engineering an external shader program developed by Boris Vorontsov

What function does it have?

It has a better indirect lighting system and the following special effects:

1. Volume Cloud (Reverse Engineering of Quant)

2. Vehicle raindrops (reverse engineering of Quant)

3. Screen Space Ambient Occlusion

4. Better performance improvement

5. Screen space color processing

6. Better rainy wetland PBR reflections

7. Advanced Manual/Auto Focus Depth of Field System

8. Ray texture of distant street lights

9. New naming system:

In ICE Shader you will get the new file format .ice added by the author
